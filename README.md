# MijiaPilotByCodexPlugin

Public Codex plugin package for using Codex with Xiaomi/Mijia smart-home devices through [`handsomejustin/mijia-control`](https://github.com/handsomejustin/mijia-control) only.

The plugin source is in:

```text
plugins/mijia-control-codex/
```

Start with:

- [Plugin README](plugins/mijia-control-codex/README.md)
- [Installation](plugins/mijia-control-codex/docs/INSTALLATION.md)
- [Configuration](plugins/mijia-control-codex/docs/CONFIGURATION.md)
- [Verification](plugins/mijia-control-codex/docs/VERIFICATION.md)
- [Security](plugins/mijia-control-codex/docs/SECURITY.md)
- [Privacy](plugins/mijia-control-codex/docs/PRIVACY.md)

This repository contains no Xiaomi account credentials, tokens, device IDs, home IDs, scene IDs, BLE bindkeys, or private smart-home configuration.
