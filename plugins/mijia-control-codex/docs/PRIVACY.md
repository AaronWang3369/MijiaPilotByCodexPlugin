# Privacy

This plugin stores no personal smart-home data.

## Not Included

The repository intentionally excludes:

- Xiaomi account usernames or passwords.
- JWT access tokens or refresh tokens.
- QR login payloads.
- device IDs.
- home IDs.
- scene IDs.
- family, room, address, or occupancy information.
- camera stream URLs or snapshots.
- BLE MAC addresses or bindkeys.

## Local Data Sources

When users run upstream `mijia-control`, the upstream service and CLI may store local data such as:

- database records for homes, devices, scenes, and users;
- token data under the upstream CLI config path;
- local `.env` files;
- logs generated by the upstream service.

Those files are outside this plugin and must be protected by the user.

## Sharing Logs

Before sharing logs or screenshots, redact:

- `Authorization` headers;
- `MIJIA_TOKEN`;
- usernames;
- QR codes;
- device IDs;
- home IDs;
- scene IDs;
- MAC addresses;
- room names and family names.
